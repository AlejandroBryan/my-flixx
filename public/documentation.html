myFlix-Api Documentation version 1

Some import to keep in mind is that you should specify api/v1/

Movies

api/v1/movies Allow you the to received movies all the available on our database.

api/v1/movies It will allow you the received a specify movie through its title or director's name. Please make sure you are add number for example: api/v1/movies/{{title}} or api/v1/movies/{{director's name}}.

api/v1/movies Make sure are adding all the require field like in the example below. Don't forget to tos send the data in json format. like example below.'

api/v1/movies/{{title}} It will allow you modified a specify movie through its title. Please make sure you are add number for example: api/v1/movies/{{title}}.

api/v1/movies/{{title}} It will allow you to delete a specify movie through its title. Please make sure you are add number for example: /api/v1/movies/{{title}}.

Users

api/v1/users Allow you the to received users all the available on our database.

api/v1/users It will allow you the received a specify user through its name. Please make sure you are add username for example: api/v1/users/{{username}}.

api/v1/users Make sure are adding all the require field like in the example below. Don't forget to send the data in json format. like example below.'

api/v1/users/{{username}} It will allow you modified a specify user through its title. Please make sure you are add number for example: api/v1/users/{{username}}.

api/v1/users/{{username}}/movies/{{movieId}} It will allow you modified a specify user favorite movies through its name. Please make sure you are add number for example: api/v1/users/{{username}}/movies/{{movieId}}.

api/v1/users/{{username}}/movies/{{movieId}} It will allow you to delete a specify user movie from his list of favorites movies Please make sure you are add number for example: api/v1/users/{{username}}/movies/{{movieId}}.

api/v1/users/{{username}} It will allow you to delete a specify user through its name. Please make sure you are add username for example: /api/v1/users/{{username}}.
